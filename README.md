# 동대문센터 업무관리

동대문센터 직원들의 출퇴근 기록과 월간 근무계획을 관리하는 웹앱입니다.

- **배포 주소**: https://ddm-1.vercel.app
- **사용 인원**: 직원 약 20여 명 + 관리자

## 주요 기능

- 출퇴근 등록 (위치 인증 포함)
- 월간 근무계획 입력 및 업무분장 관리
- 근무 캘린더 (날짜별 계획/출근 현황)
- 일정 변경 요청 및 관리자 승인
- 관리자 대시보드 (오늘현황, 전체일정, 출퇴근 매트릭스, 직원관리)
- CSV 다운로드 (출퇴근 기록, 전체일정, 계획 현황)
- 모바일 앱 설치 지원 (PWA)

## 사용 기술

- `index.html` 단일 파일 (HTML/CSS/JavaScript)
- [Supabase](https://supabase.com) — 데이터 저장
- [Vercel](https://vercel.com) — 배포
- Claude(AI) — 개발 및 유지보수

## 파일 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 전체 코드 |
| `manifest.json` | PWA 앱 설정 (이름, 아이콘) |
| `sw.js` | 서비스워커 (앱 설치 지원) |
| `icon-192.png`, `icon-512.png` | 앱 아이콘 |
